# **Automação de Indicadores**

## **Objetivo**  
Treinar e criar um Projeto Completo que envolva a automatização de um processo feito no computador.

## **Descrição**  

Imagine que você trabalha em uma grande rede de lojas de roupa com 25 lojas espalhadas por todo o Brasil.  

Todo dia, pela manhã, a equipe de análise de dados calcula os chamados **One Pages** e envia para o gerente de cada loja o resumo com os principais indicadores da sua unidade.  

Um **One Page** é um relatório simples e direto ao ponto, permitindo que os gerentes comparem os resultados entre as lojas e avaliem o desempenho da unidade em relação às metas diárias.

### **Automação**  
O objetivo do projeto é automatizar esse processo, garantindo que:  

✅ Os indicadores sejam calculados automaticamente.  
✅ Um e-mail seja enviado para cada gerente com seu OnePage no corpo do e-mail.  
✅ O arquivo completo com os dados da loja seja enviado em anexo.  

## **Tecnologias Utilizadas**  
- Python  
- Pandas  
- OpenPyXL  
- Smtplib
